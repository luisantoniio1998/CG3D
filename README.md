# CG3D

Game developed for Computer Graphics Course Evaluation using Java and Java3D API.

## Technologies
- Java
- Java3D API
- Eclipse IDE

## Features
- 3D rendering
- Camera controls
- Object manipulation
- Lighting effects
- Collision detection

## Setup
1. Install Java + Java3D
2. Clone repository
3. Open in Eclipse
4. Run Main.java
